# PDF Merger Desktop App

A simple desktop application to merge multiple PDF files into a single PDF.

## Features

- Merge multiple PDFs
- Select output location
- Simple desktop GUI
- Windows executable support
- Built using Python

## Tech Stack

- Python
- Tkinter
- pypdf
- PyInstaller
